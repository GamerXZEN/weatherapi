Weather Data API Home

This is the home for the Weather API built by GamerXZEN using the REST API Architecture

URL format: http://127.0.0.1:1472/v2.1/station

URL example: http://127.0.0.1:1472/v2.1/10/

URL format: http://127.0.0.1:1472/v2.1/station/year/year

URL example: http://127.0.0.1:1472/v2.1/10/year/2020

URL format: http://127.0.0.1:1472/v2.1/station/date/date

URL example: http://127.0.0.1:1472/v2.1/10/date/2020-10-29
